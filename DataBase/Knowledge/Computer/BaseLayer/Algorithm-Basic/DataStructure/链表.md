#dataStructure

链表和[[Array]]是最基础的数据结构，两者的区别在于数组是连续的内存空间，链表是离散的内存空间。因此数组可以随机访问，而链表则不能。
# 常见实现
```javaScript
// 单链表
interface ListNode<T> {
	data: T;
	next: ListNode<T>;
}

// 双向链表
interface ListNode<T> {
	data: T;
	pre: ListNode<T>;
	next: ListNode<T>;
}
```
# 常见操作
## 单链表反转
```kotlin
fun reverseList(head: ListNode, tail: ListNode) {  
    tail.next = null  
    var pre = head  
    var tempNode: ListNode? = head.next ?: return  
    head.next = null  
    while (tempNode != null) {  
        val next = tempNode.next  
        tempNode.next = pre  
        pre = tempNode  
        tempNode = next  
    }  
}
```
