# 问题描述

-   1个整数数组中，有1个数字出现1次，其他数字都出现3次
-   请找出这个出现1次的数字

> 1.  额外要求：时间复杂度 = O（n）、空间复杂度 = O（1）
> 2.  如，输入数组 = { 2,4,2,5,2,5,5 }，则输出4
