## 开放定址法

## 再Hash法
设计多个哈希函数
## 链地址法
比如通过链表将同一 index 的元素串起来
# JDK1.8的解决方法
-   默认使用**单向链表**将元素串起来
-   在添加元素时，可能会由**单向链表**转为**红黑树**来存储元素  
    比如当哈希表容量 ≥ 64 且 单向链表的节点数量大于 8 时
-   当**红黑树**节点数量少到一定程度时，又会转为**单向链表**
-   JDK1.8 中的哈希表是使用 **链表+红黑树** 解决哈希冲突
