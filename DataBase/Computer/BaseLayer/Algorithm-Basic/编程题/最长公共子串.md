题目：求两个字符串的最长公共子串长度

-   ABCBA 和 BABCA 的最长公共子串是 ABC，长度为 3

假设 2 个字符串分别是 str1、str2：



-   i ∈ [1, str1.length]
-   j ∈ [1, str2.length]
假设dp(i,j) 是以 str1[i – 1]、str2[j – 1] 结尾的最长公共子串长度：

- dp(i,0)、dp(0,j) 初始值均为 0
- 如果 str1[i – 1] = str2[j – 1]，那么
- dp(i, j) = dp(i – 1, j – 1) + 1
- 如果 str1[i – 1] ≠ str2[j – 1]，那么 
dp(i, j) = 0


-   空间复杂度：O(n ∗ m)
-   时间复杂度：O(n ∗ m)
