# 基本概念
二叉树是一种有序[[Trees|树]]。每个节点的度是2。
n0、n1、n2分别为度为0、1、2的结点的个数，有：
$$
n_{1} + 2 * n_{2} = n_{0}+ n_{1}+ n_{2}
$$
$$
n_{0}= n_{2} + 1
$$
**真二叉树**：所有节点的度要么为0，要么为2.
